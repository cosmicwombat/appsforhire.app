New Customer Checklist
1
Run setup script — python3 scripts/new_customer.py from your app_for_hire folder. Creates GitHub repo, pushes template, enables Pages.
2
Cloudflare DNS — Add CNAME record: [slug] → cosmicwombat.github.io with proxy ON (orange cloud).
3
Cloudflare Access — Zero Trust → Access controls → Applications → Add Self-hosted app. Add policy in the Policies tab with client's email. ⚠️ Policy must be inside the app, not standalone.
4
GitHub Pages — Set custom domain to [slug].appsforhire.app and enforce HTTPS in the repo Settings → Pages.
5
Send Stripe link — Email the client their hosting subscription link and confirm the app is working end-to-end before billing starts.
6
Push data update — Run python3 scripts/update_admin_data.py then git add . && git commit -m "Add [client] customer data" && git push to update this dashboard.
