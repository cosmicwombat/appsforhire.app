New Customer Workflow
1
Run new_build.py — python3 scripts/new_build.py — sets up the build folder and prints the Cowork prompt. Commit builds/ after.
2
Open a new Cowork session — paste the generated prompt. Claude builds the app in builds/{slug}/. Review, tweak, done.
3
Publish from the Builds tab — connect GitHub (token), find the build, click Publish. Creates the client repo and enables Pages automatically.
4
Add Cloudflare DNS — CNAME {slug} → cosmicwombat.github.io, Proxy ON. Cloudflare Access policy with client email.
5
Custom/Pro only — register client API keys: curl -X POST worker.appsforhire.app/admin/set-client-keys.
6
Message the client — ✉️ button in Customers sends the "App is Live" email with their URL.
